Overview

This project applied system dynamics modelling to analyse complex system behaviour using causal loop diagrams (CLDs) built in STELLA. The goal was to identify feedback loops, understand how variables interact over time, and generate insights to support decision-making.

What I Did

Modelling

  • Mapped reinforcing and balancing feedback loops
  • Analysed system behaviour over time
Analysis
  • Identified leverage points for intervention
  • Traced unintended consequences through feedback
  • Documented system archetypes
  • Generated decision-making recommendations
